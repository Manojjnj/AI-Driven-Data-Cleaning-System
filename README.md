# AI-Driven Data Cleaning System

[![Streamlit](https://img.shields.io/badge/Streamlit-App-green)](https://share.streamlit.io/yourusername/ai-driven-data-cleaning-system)

## Overview
This project is an **AI-powered data cleaning tool** built with Streamlit. It allows users to:

- Upload raw datasets
- Automatically clean data (handle missing values, duplicates, etc.)
- Generate a **comprehensive data cleaning report**
- Download the cleaned data and summary report
- Optionally provide voice instructions for dynamic cleaning using OpenAI API
